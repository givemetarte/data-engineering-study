### 풀이법

- 전형적인 다이나믹 프로그래밍 문제 
- 아직 잘 모르겠어서 일단 외움...!

```py
n = int(input())
dp = [0] * (n+1)

for i in range(2, n+1):
    # 1을 빼주는 경우
    dp[i] = dp[i-1] + 1

    if (i % 2) == 0:
        dp[i] = min(dp[i], dp[i//2]+1)
    if (i % 3) == 0:
        dp[i] = min(dp[i], dp[i//3]+1)

print(dp[n])
```
